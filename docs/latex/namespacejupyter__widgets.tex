\doxysubsubsection{Detailed Description}
A file for displaying widgets in the Jupyter notebook. 

This file containts widgets for sepecifying options in the Jupyter notebook using the ipywidgets library (\href{https://github.com/jupyter-widgets/ipywidgets}{\texttt{ https\+://github.\+com/jupyter-\/widgets/ipywidgets}}). Using Jupyter notebook is not necessary for using the program, as the program can be run in as a python script. However, the widgets provide a graphical user interface for making input files, running the code, and displaying the results. The Jupyter notebook can be run locally or on the clouds using Binder (\href{https://mybinder.org/v2/gh/alenaizan/pnab/master?filepath=binder}{\texttt{ https\+://mybinder.\+org/v2/gh/alenaizan/pnab/master?filepath=binder}}).

The widgets have four main components\+: Backbone, Bases, Helical Parameters, and Runtime Parameters. The Bases section does not require the user input but displays some information about the available nucleobases in the library. The N\+G\+L\+View library is used for visualization

To run the widgets from a Jupyter notebook, simply execute the following\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{import} pnab}
\DoxyCodeLine{pnab.builder()}
\end{DoxyCode}

Display search parameters based on the chosen algorithm. 

There are six search algorithms and each one has a set of input parameters. The search algorithms are\+:


\begin{DoxyItemize}
\item Systematic Search\+: Requires specifying the dihedral angle step size, {\itshape dihedral\+\_\+step}.
\item Monte Carlo Search\+: Requires specifying the number of steps, {\itshape num\+\_\+steps}, and the Monte Carlo temperature, {\itshape monte\+\_\+carlo\+\_\+temperature}.
\item Weighted Monte Carlo Search\+: Also requires specifying the weighting temperature, {\itshape weighting\+\_\+temperature}.
\item Random Search\+: Requires specifying the number of steps, {\itshape num\+\_\+steps}.
\item Weighted Random Search\+: Also requires specifying the weighting temperature, {\itshape weighting\+\_\+temperature}.
\item Genetic Algorithm Search\+: Requires specifying the number of generations, {\itshape num\+\_\+steps}, the population size, {\itshape population\+\_\+size}, the mutation\+\_\+rate, {\itshape mutation\+\_\+rate}, and the crossover rate, {\itshape crossover\+\_\+rate}.
